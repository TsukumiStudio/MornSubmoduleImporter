# MornSubmoduleImporter

## 概要

Gitサブモジュールを UI から管理するエディタツール。

## 依存関係

| 種別 | 名前 |
|------|------|
| 外部パッケージ | なし |
| Mornライブラリ | なし |

## 使い方

### エディタウィンドウを開く

`Tools > Morn Submodule Importer` から開きます。

### サブモジュールの追加

1. `submodule.txt` にリポジトリURLを列挙
2. UI上で追加したいサブモジュールを選択
3. 「Add Selected Submodules」ボタンでインストール

### 機能

- インストール状態をUIで視覚化表示
- git submoduleコマンドを自動実行
